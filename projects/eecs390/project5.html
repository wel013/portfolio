Project 5: Code Generation for uC Language in C++

uC language is a language developped by the professcor of this class for the use in this class.

Overview

The focus of this project was to implement a code generator for the uC language, a small language within the C family, to generate C++ code. The primary goals were to grasp metaprogramming concepts, understand language translation processes, and gain proficiency in using macros and templates. Additionally, it aimed to provide experience in working with a complex codebase, particularly dealing with code written by someone else.

Key Components

  • Phase 1: Generating Type Declarations: The initial step involved generating forward declarations for user-defined types and functions in order to support usage before definition, adhering to the semantics of uC. Prioritization was given to forward declarations of types preceding those of functions.
  • Phase 2: Generating Function Declarations: This phase focused on generating forward declarations specifically for user-defined functions.
  • Phase 3: Generating Type Definitions: Following the order constraints in C++, this phase concentrated on generating full definitions for each user-defined type. This ensured that the type contents were accessible before function definitions.
  • Phase 4: Generating Function Definitions: Subsequently, full definitions for each user-defined function were generated, marking the completion of the code generation process for function implementations.
  • Phase 5: Polymorphic Operations: The final phase aimed to implement operations involving polymorphism, specifically the addition of two items. Due to the absence of type computation from a prior project, reliance was placed on the target C++ compiler to resolve these operations.

Achievements

  • Successfully implemented a code generator for uC language constructs, including type declarations, function declarations, type definitions, and function definitions.
  • Demonstrated the application of macros, templates, and language translation techniques in generating C++ code from the uC language.
