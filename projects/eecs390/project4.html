Project 4: Semantic Analysis for uC Language

uC language is a language developped by the professcor of this class for the use in this class.

Overview

This project involved implementing a semantic analyzer for the uC language, a small C family language used specifically within the course. The primary objectives were to write a substantial object-oriented Python code, emphasizing inheritance, modularity, and code reuse. It aimed to provide insight into compiler workings, particularly the semantic analysis phase, which verifies the correctness of the program structure.

Key Components

  • Phase 1: Finding Declarations: The initial phase focused on collecting type and function declarations, enabling support for the usage of types and functions before their actual definition.
  • Phase 2: Resolving Types: Subsequently, the task was to map occurrences of type names to the corresponding Type object, facilitating accurate type resolution.
  • Phase 3: Resolving Function Calls: This phase concentrated on matching function calls with the appropriate Function object being invoked, ensuring accurate function call resolution.
  • Phase 4: Checking Fields and Variables, Resolving Names: The implementation involved constructing VarEnv objects representing user-defined type or function scopes, ensuring distinct names for fields, parameters, and variables within each scope while recording their types.
  • Phase 5: Checking Basic Control Flow: The focus was on verifying the appropriate usage of break and continue statements within loops, leveraging contextual information to determine their validity.
  • Phase 6: Computing and Checking Types: This phase involved computing the types of all expressions and validating the usage of types in the appropriate contexts, ensuring the type attribute of the AST nodes reflected the computed type.

Achievements

  • Acquired proficiency in understanding and extending a sizable codebase authored by others, focusing on relevant functionalities essential for the semantic analysis phase.
  • Leveraged inheritance, polymorphism, and modularity within a large class hierarchy to prevent code duplication and enhance maintainability.

Technologies & Skills:

Python, Codebase Navigation, Abstract Syntax Trees (AST), Program Analysis Basics.
