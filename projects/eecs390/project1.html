Project 1: Graph Library and PageRank Implementation

Overview

This project centered on constructing a fundamental graph library in Python 3 and implementing a simplified iteration of the powerful PageRank algorithm used extensively in search-engine optimization.

Key Components

  • Graph ADTs Development: Created a robust set of classes encompassing essential methods vital for graph representation. This included the implementation of Node, Edge, BaseGraph, UndirectedGraph, DirectedGraph, and efficient error handling with GraphError.
  • PageRank Algorithm Implementation: Implemented the PageRank algorithm leveraging the established abstract data types (ADTs) developed earlier in the graph library phase.
  • The implemented PageRank algorithm exhibited a complexity of O(NlogN + ElogE + m(N+E)), where 'm' signifies the number of iterations required.

Achievements

  • Developed a comprehensive graph library facilitating the representation and manipulation of graphs in Python.
  • Successfully implemented the PageRank algorithm.

Technologies & Skills:

Python3, object-oriented programming
