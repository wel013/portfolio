Project 4: MapReduce Framework Implementation

Overview

Implemented a custom MapReduce framework in Python, mirroring Google's MapReduce paper. This framework enables the execution of MapReduce programs through distributed processing on a cluster of computers, such as AWS EMR.

Key Components

  • Worker Registration and Management:
    • Manager tracks available Workers for distributing tasks among ready Workers.
  • Job Execution and Management:
    • New job requests are received by the Manager and added to the queue. Manager executes jobs in sequence and waits until completion before initiating the next job.
  • Input Partitioning and Mapping:
    • Manager scans input directories, dividing files into partitions.
    • Mapping tasks are assigned to Workers, managing tasks' order based on Worker availability.
    • Workers execute map executables on specified input files, sort output, and share temporary data with the Manager.
  • Reducing and Output Processing:
    • Reduce tasks are allocated to Workers during the Reduce Stage, ordered by registration time.
    • Workers merge and process input files, running reduce executables and storing output in the final directory specified by the Manager.
  • Fault Tolerance:
    • Mechanism to handle Worker failures: If a Worker fails to respond for a defined period, reassign its pending work to another available Worker.

Achievements

  • Framework Development: Engineered a robust MapReduce framework from scratch in Python, adhering to Google's original MapReduce concepts. The framework facilitated the execution of MapReduce programs, demonstrating proficiency in distributed computing paradigms.
  • Fault-Tolerant Design: Implemented fault tolerance mechanisms to handle Worker failures within the distributed system. The framework dynamically reassigned pending tasks if Workers became unresponsive, ensuring uninterrupted job execution.

Technologies & Skills:

Python, MapReduce, distributed systems, fault tolerance, concurrency (threads and processes) and Network Communication (sockets).
