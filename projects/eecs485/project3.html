Project 3: Client-Side Dynamic Instagram Clone with REST API Integration

Overview

Developed an advanced Instagram clone utilising client-side dynamic pages and a REST API, fostering a seamless and interactive user experience.

Key Components

  • Database Operations: Created shell scripts for managing the database, including functionalities for creation, destruction, resetting, and displaying database contents.
  • REST API Development: Defined and implemented specific endpoints for retrieving information, managing posts, likes, and comments
  • HTTP Basic Access Authentication: Implemented HTTP basic authentication requiring a username and password, enhancing security measures.
  • Client-Side Web Pages: Developed interactive web pages using JavaScript, enabling functionalities such as timestamped posts, real-time addition/deletion of likes and comments, and dynamic content loading through scrolling.

Achievements

  • Successfully developed an Instagram clone with a responsive and interactive user interface powered by client-side dynamic pages and a RESTful API.
  • Acquired comprehensive knowledge and hands-on experience in JavaScript, AJAX, RESTful API implementation, and client-server interaction.

Technologies & Skills:

HTML, CSS, Python programming, shell scripting, JavaScript, React Components, AJAX, REST API, client-server interaction.
