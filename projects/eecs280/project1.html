Project 2: Content-Aware Image Resizing with Seam Carving

Overview

Building an image resizing program utilizing a seam-carving algorithm.

Key Components

  • Matrix Module:
    • Developed a Matrix Abstract Data Type (ADT) to represent matrices containing integers. The matrix struct was defined to specify the interface for functions operating on matrices. The module involved allocating storage and using an initializer function. Access to individual elements in a matrix was facilitated through pointers retrieved using Matrix_at. The module emphasized efficient memory handling and provided a straightforward interface for matrix manipulation.
  • Image Module:
    • Created an Image ADT, similar to a Matrix but designed to contain Pixels instead of integers. Each Pixel included three integers representing red, green, and blue (RGB) color components. The Pixel type was considered "Plain Old Data" (POD), allowing direct use of its member variables without a separate interface. The module highlighted the importance of adhering to a well-defined interface while managing image data efficiently.
  • Processing Module:
    • The processing module comprised functions for various image processing operations, with a focus on content-aware resizing using the seam-carving algorithm. Key functions included seam_carve, seam_carve_width, and seam_carve_height, serving as the main interfaces for resizing. These functions utilized the seam carving algorithm to shrink either the image's width or height in a content-aware manner. The module involved calculating energy and cost matrices and identifying minimal seams for removal.

Achievements

  • Implemented Matrix and Image ADTs, providing a robust and well-defined interface for working with matrices and images.
  • Developed an effective content-aware resizing algorithm using seam carving, allowing for context-aware adjustments of image width and height.

Technologies & Skills:

C++, Testing, Debugging, pointers, array, string, stream IO, and Abstract Data Types.
