Project 5: Piazza Post Topic Classifier

Overview

Project 5 aims to automatically identify the topic of posts from the EECS 280 Piazza platform utilizing Natural Language Processing (NLP) and Machine Learning techniques.

Key Components

  • Binary Search Tree ADT:
    • Implemented a Binary Search Tree (BST) Abstract Data Type (ADT) with templated parameters, catering to the specific requirements of the project. The BST included nodes representing elements and pointers to left and right subtrees. Special emphasis was placed on overriding comparators to facilitate correct ordering within the tree.
  • Map ADT:
    • Developed a Map ADT, an associative container containing key-value pairs, akin to std::map. This ADT replicated functionalities such as insertion, retrieval (find), and deletion. The map provided an essential structure for subsequent tasks in the project.
  • Classifier Model:
    • Implemented a classifier model that utilized a bag of words trained on a given dataset and conditional probability to classify the subject of a post from the EECS 280 Piazza platform. This involved leveraging NLP and machine learning techniques to automate the subject identification process.

Achievements

  • Successfully implemented a Binary Search Tree ADT with templated parameters and specialized comparators for accurate ordering.
  • Implemented a Map ADT, replicating functionalities of std::map, providing a structured associative container for key-value pairs.
  • Implemented a classifier model utilizing NLP and machine learning techniques, showcasing practical application in subject identification from Piazza posts.

Technologies & Skills:

Function Objects and Recursion, Container ADTs, Dynamic Memory, The Big Three, Linked Lists, iterators, Binary Tree, Templates, Map data structure.
