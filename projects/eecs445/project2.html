Project 2: Landmark Classification in Europe using Convolutional Neural Networks (CNN)

Overview

Implemented a Convolutional Neural Network (CNN) to classify famous landmarks in Europe using image data.

Key Components

  • Baseline Model for Binary Prediction: Established an initial model for Pantheon and the Hofburg Imperial Palace.
  • Challenges and Improvements: Enhanced the model's accuracy on a more challenging dataset and explored advanced techniques to improve predictions.
Data Preprocessing
  • Normalised image data to create a standardised and effective dataset.
  • Implemented a CNN architecture comprising convolutional layers, pooling layers, and fully connected layers for classification.
Model Specifications:
  • Utilised Crossentropy loss, adam optimizer, SAME padding, and MAX pooling layers for model optimization.
  • Employed Gradient-weighted Class Activation Mapping (Grad-CAM) to interpret the model's learned features and ensure interpretability.
  • Incorporated Transfer Learning using Colosseum, Rialto Bridge, Museu Nacional d'Art de Catalunya, Petronas Towers, Berlin Cathedral, Hagia Sophia, Gaud Casa Batllo in Barcelona, and St Stephen's Cathedral in Vienna to enhance the model's performance on additional landmarks.
Challenge Techniques:
  • Explored diverse data augmentation methods like grayscale transformation, rotation, and edge detection.
  • Experimented with VGG16 and various ResNet architectures, performing hyperparameter tuning (e.g., weight decay, learning rate, batch size) for improved performance.
  • Implemented feature selection techniques to optimise the model's learning process.

Achievements

  • Base model achieved an accuracy of approximately 96%, while the challenge mode achieved an accuracy of around 88%.
  • Successfully visualised the model's knowledge using Grad-CAM, ensuring its interpretability.
  • Explored and implemented various advanced CNN architectures and data augmentation strategies to improve accuracy.

Technologies & Skills:

Python, TensorFlow/Keras, CNN, Transfer Learning, Data Augmentation, Model Evaluation, Feature Selection.

Takeaways

  • Profound understanding and application of CNNs for image classification tasks.
  • Experience in model interpretation using visualisation techniques like Grad-CAM.
  • Expertise in hyperparameter tuning and optimising CNN architectures for improved performance.
