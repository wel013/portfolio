Project 1: Sentiment Analysis on Movie Reviews using SVM

Overview

Utilised review data from Prime Video's extensive movie catalogue to develop a sentiment analysis model using Support Vector Machines (SVM).

Key Components

  • Baseline Model for Binary Prediction: Constructed an initial model for binary sentiment prediction.
  • Challenges and Improvements: Implemented enhancements for multiclass prediction and explored various feature engineering techniques to refine predictions.
Feature Extraction
  • Employed binary encoding for word presence indication.
  • Conducted extensive hyperparameter tuning, including kernel selection (linear and quadratic), L1/L2 penalty, and different loss functions.
  • Explored asymmetric cost functions to tackle class imbalances in training data.
Exploratory Data Analysis (EDA) and Bias Identification
  • Utilised word embedding and Word Embedding Association Test (WEAT) to uncover biases and assess word associations.
  • Investigated techniques to mitigate bias effects, including preserving punctuation for added context and applying stop word removal and lemmatization for dimensionality reduction. .
Advanced Feature Engineering:
  • Leveraged CountVectorizer to extract unigrams, bigrams, and trigrams as features.
  • Incorporated Term Frequency-Inverse Document Frequency (TF-IDF) for n-gram feature representation.
  • Explored one-vs-all and one-vs-one multiclass strategies for extended classification scenarios.

Achievements

  • Baseline model was able to achieve an accuracy of ~0.91 on the binary testing data.
  • Challenge model was able to achieve an accuracy of ~0.87 on the multiclass testing data.
  • Addressed bias concerns and class imbalances, enhancing the robustness of the sentiment analysis model.

Technologies & Skills:

Python, Scikit-learn, SVM, NLP, Data Analysis, Feature Engineering, Model Evaluation, Hyperparameter Optimization.

Takeaways

  • Gained expertise in SVMs and their application in sentiment analysis.
  • Developed proficiency in feature extraction, hyperparameter tuning, and bias mitigation techniques.
