Project 3: SillyQL

Overview

The project creates a relational database emulator with a command-line interface based on a subset of a standard query language using C++.

Key Components

  • Database Shell:
    • Create a user-friendly shell allowing interactions with a single database.
  • Implementation of Database Commands:
    • Create: Command to create a new database.
    • QUIT: Command to exit the program and destroy the in-memory database.
    • REMOVE: Command to remove specific table from the database.
  • Implementation of Table Commands:
    • Insert: Command to insert data into the database.
    • Print: Command to display the contents of the database.
    • DELETE: Command to delete a specified element from the table.
    • JOIN: Command to perform a join operation between tables in the database.
    • Generate: Command to generate specific data within the database.

Achievements

  • Implementation of a relational database emulator with a subset of SQL commands.
  • Proficient selection and utilization of appropriate data structures (Vector, Unordered_map, Map) for database operations.
  • Design and implementation of efficient algorithms for various database functionalities.

Technologies & Skills:

C++, Database Emulation, Data Structures (Vector, Unordered_map, Map), Algorithm Design, Command-Line Interface.
