Project : Among Us Spaceship Routing Algorithm

Overview

This project applied graph algorithms within the multiplayer murder mystery game Among Us. The project involves playing the roles of both an Impostor and a ghost Crewmate in a spaceship environment, aiming to identify efficient routes and strategically contribute to winning the game.

Key Components

  • Minimum Spanning Tree (MST) - Vent Network:
    • Devise a vent network (MST) that efficiently connects every room while minimizing the total distance of vents needed.
  • Travelling Salesperson Problem (TSP) Approximation:
    • Research and implement an algorithm to approximate a solution to the TSP, ensuring it is sufficiently close and fast.
  • Optimal TSP Solution with Branch and Bound (BnB) Algorithm:
    • Implement a solution to the TSP that guarantees optimality.

Achievements

  • Successful implementation of MST, Travelling Saleman algorithms with and without heuristics.

Technologies & Skills:

Graph Algorithms, Minimum Spanning Tree (MST), Travelling Salesperson Problem (TSP), Heuristic Approaches, Branch and Bound Algorithm.
