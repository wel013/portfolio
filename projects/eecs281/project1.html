Project 1: Rescue the Countess

Overview

Project 1 involves implementing fundamental pathfinding algorithms in C++ to guide Marco through a maze-like castle to rescue Countess Cherry. The program offers both queue-based and stack-based routing schemes to navigate the castle efficiently.

Key Components

  • Reading in Castle Layout (Input Modes):

    • Map Input Mode (M) outputs an updated map of the castle rooms, incorporating characters to visualize the path chosen by Marco during the rescue mission.
  • Pathfinding Algorithms:
    • The program incorporates both queue-based and stack-based pathfinding algorithms to dynamically navigate the castle layout and reach Countess Cherry. These algorithms ensure optimal routes while considering obstacles and warp pipes.
    • The pathfinding algorithm should work regardless which input mode is given and should generate the same path for both output modes.
