Project 3: CPU Cache Integration into Behavioral Simulator

Overview

Focused on understanding cache design principles and integrating a simulated CPU cache (unified instruction/data) into a behavioral simulator. The aim was to comprehend address referencing, cache operations, and their integration within a processor simulator.

Key Components

  • CPU Cache Simulation: Simulated a unified instruction/data CPU cache to handle accesses while executing assembly-language programs within the processor simulator.

Achievements

  • Successfully simulated a unified instruction/data CPU cache and seamlessly integrated it into an existing behavioral simulator.
  • Gained practical experience in managing memory accesses through a simulated cache while executing assembly-language programs in the processor simulator.

Technologies & Skills:

C, assembly language, cache.
