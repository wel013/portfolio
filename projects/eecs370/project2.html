Project 2: LC-2K Pipelined Behavioral Simulator

Overview

Developed a cycle-accurate behavioral simulator for a pipelined implementation of the LC-2K architecture. The objective was to understand and implement pipelining concepts along with data forwarding, branch prediction, and handling various hazards.

Key Components

  • Cycle-Accurate Behavioral Simulator: Created a detailed simulator for a pipelined LC-2K implementation, incorporating data forwarding and branch prediction mechanisms.
  • Handling Data Hazards: Managed two types of data hazards:
    • Data hazards resolved through data forwarding without stalling the pipeline.
    • Data hazards causing stalls in the pipeline while ensuring data forwarding.
  • Control Hazard and Branch Prediction: Implemented branch prediction by speculating branch-not-taken, deciding branch outcomes during the MEM stage, and discarding incorrect predictions.

Achievements

  • Successfully developed a cycle-accurate behavioral simulator demonstrating pipelined LC-2K execution with data forwarding and branch prediction.
  • Gained hands-on experience in identifying, managing, and resolving various hazards encountered in pipelined architectures.

Technologies & Skills:
