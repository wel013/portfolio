Project 1: LC2K Assembly Program and Simulator

Overview

Designed and implemented an LC2K assembly program and simulator with the objectives of translating assembly-language programs to machine language, simulating program execution, and creating a multiplication assembly program.

Key Components

  • Translation to Machine Language: Developed a program to translate assembly-language programs into 32-bit machine language instructions, including opcode and numeric equivalence.
  • Simulation: Built an LC2K simulator that executed the program, displaying the state of the program until a halt instruction was encountered.
  • Multiplication Assembly: Created an LC2K assembly program to efficiently multiply two positive 15-bit numbers within the LC2K architecture.

Achievements

  • Successfully implemented a translator for assembly-language programs into 32-bit machine language.
  • Built an LC2K simulator to execute programs and demonstrated the ability to create an assembly program for multiplying positive 15-bit numbers within the LC2K architecture.

Technologies & Skills:

C, assembly language, binary.
